# 仿天眼查项目

#技术栈
```
前端：vue3+Webpack+Typescript+Elementplus+less
后端：Springboot+Redis+Mysql8
```

## 运行准备
```
前端 yarn install
后端 文件./backEnd 要求java8+ 启动redis mysql导入文件（数据库文件在管理系统中）
```
### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
